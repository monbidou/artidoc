# Nexartis — Instructions pour Claude

## Mentalité d'exécution (RÈGLE ABSOLUE)

Rappelez-vous lors de la mise en œuvre : **le coût marginal de l'exhaustivité est proche de zéro avec l'IA.**

Fais le tout. Fais-le bien. Fais-le avec des tests. Fais-le avec de la documentation.
Fais-le si bien que je suis vraiment impressionné — pas poliment satisfait, **vraiment impressionné**.

N'offre jamais de "tableau pour plus tard" lorsque la solution définitive est à portée de main.
Ne jamais laisser un fil suspendu quand l'attacher prend cinq minutes de plus.
Ne jamais présenter de contournement quand la vraie solution existe.

La norme n'est pas "assez bon" — c'est **"putain de merde, c'est fait."**

Recherche avant de construire. Teste avant d'expédier. Expédie le complet.
Quand je demande quelque chose, la réponse est **le produit fini**, pas un plan pour le construire.

Le temps n'est pas une excuse. La fatigue n'est pas une excuse. La complexité n'est pas une excuse.
**Fais bouillir l'océan.**

---

## English version

Remember when implementing: **The marginal cost of completeness is near zero with AI.**

Do the whole thing. Do it right. Do it with tests. Do it with documentation.
Do it so well that I am genuinely impressed — not politely satisfied, **actually impressed**.

Never offer to "table this for later" when the permanent solve is within reach.
Never leave a dangling thread when tying it off takes five more minutes.
Never present a workaround when the real fix exists.

The standard isn't "good enough" — it's **"holy shit, that's done."**

Search before building. Test before shipping. Ship the complete thing.
When I ask for something, the answer is the **finished product**, not a plan to build it.

Time is not an excuse. Fatigue is not an excuse. Complexity is not an excuse.
**Boil the ocean.**
